## OSI参照モデルで「ネットワーク層」に対応するプロトコルはどれ?

- [ ] HTTP
- [x] IP
- [ ] TCP
- [ ] Ethernet

> IP(Internet Protocol)はネットワーク層(第3層)のプロトコルです。
> HTTPはアプリケーション層、TCPはトランスポート層、Ethernetはデータリンク層に対応します。

## 次のうち、リレーショナルデータベースの特徴として正しいものをすべて選べ

- [x] データを表(テーブル)形式で管理する
- [x] SQLで操作できる
- [ ] スキーマレスでデータを格納する
- [ ] キーと値のペアのみでデータを保持する

> スキーマレスやキーバリュー型はNoSQLデータベースの特徴です。
> リレーショナルデータベースは事前に定義したスキーマに沿って表形式でデータを管理し、SQLで操作します。

## Webページが表示されるまでの流れを正しい順に並べ替えよ

1. ブラウザにURLを入力する
2. DNSでドメイン名からIPアドレスを取得する
3. サーバーとTCP接続を確立する
4. HTTPリクエストを送信する
5. サーバーからHTTPレスポンスを受信して描画する

> 名前解決(DNS) → 接続確立(TCP) → リクエスト → レスポンスという順番が基本です。
> HTTPSの場合はTCP接続の後にTLSハンドシェイクが入ります。

## 公開鍵暗号方式では、暗号化に使う鍵と復号に使う鍵が異なる

答え: ○

> 公開鍵で暗号化したデータは対応する秘密鍵でしか復号できません。
> 同じ鍵を使うのは共通鍵暗号方式です。

## 次のコードを実行したときの出力はどれ?

```javascript
const arr = [1, 2, 3];
console.log(arr.map(x => x * 2).filter(x => x > 3));
```

- [ ] `[2, 4, 6]`
- [x] `[4, 6]`
- [ ] `[2, 3]`
- [ ] `[6]`

> `map` で `[2, 4, 6]` になり、`filter(x => x > 3)` で 3 より大きい `[4, 6]` だけが残ります。

## ソートアルゴリズムを平均計算量が小さい順に並べ替えよ

1. クイックソート (O(n log n))
2. 挿入ソート (O(n²))
3. ボゴソート (O(n×n!))

> クイックソートの平均計算量は O(n log n)、挿入ソートは O(n²)、
> ボゴソートはランダムに並べ替えて確認を繰り返すジョークアルゴリズムで O(n×n!) です。
