# 🚃 Learning Express

Markdown ファイルを元データにした、GitHub Pages で動く学習クイズアプリです。
ビルド不要・依存ライブラリなしの静的サイトなので、**md ファイルを追加して push するだけ**で問題集が増えます。

## できること

- **四択(択一)問題** — 選択肢をタップで即judge
- **複数選択問題** — `[x]` を複数付けるだけで「すべて選べ」形式に
- **並び替え問題** — ▲▼ボタンで順番を組み立てて回答
- **正誤(○×)問題** — ○/× の2択
- 回答ごとの**正誤判定と解説表示**
- 結果画面(スコア・所要時間・全問の振り返り)
- **間違えた問題だけを復習**するモード(履歴はブラウザの localStorage に保存)
- 出題シャッフル/選択肢の自動シャッフル
- デッキごとの成績記録(挑戦回数・前回・自己ベスト)
- ダークモード、キーボードショートカット(`1`〜`4` で選択、`Enter` で次へ、○×は `o` / `x`)
- スマホ対応レイアウト

## 公開手順(初回のみ)

1. このブランチを `main` にマージする
2. リポジトリの **Settings → Pages → Build and deployment → Source** を **GitHub Actions** にする
3. `main` に push されるたびに自動デプロイされ、`https://<ユーザー名>.github.io/Learningexpress/` で公開されます

## 問題の追加方法

1. `data/` に Markdown ファイルを作る(下のフォーマット参照)
2. `data/index.json` の `decks` 配列にファイルパスを追記する

```json
{
  "decks": [
    "data/sample-it.md",
    "data/my-new-deck.md"
  ]
}
```

3. commit & push → 自動デプロイ

> ローカルで確認する場合は `python3 -m http.server` などで配信して `http://localhost:8000` を開いてください(`file://` 直開きでは fetch が動きません)。

## Markdown フォーマット

1ファイル = 1デッキ。`#` がデッキ名、`##` が問題の区切りです。
問題形式は書き方から自動判定されます。サンプルは [`data/sample-it.md`](data/sample-it.md) を参照してください。

````markdown
# デッキのタイトル

ここに書いた文章はデッキの説明になります。

## 四択問題の問題文をここに書く

- [x] 正解の選択肢
- [ ] 不正解の選択肢
- [ ] 不正解の選択肢
- [ ] 不正解の選択肢

> ここに解説を書く(回答後に表示。省略可)

## 複数選択は [x] を2つ以上付けるだけ

- [x] 正解その1
- [x] 正解その2
- [ ] 不正解

## 並び替え問題は番号付きリストを「正解の順番」で書く

1. 最初にやること
2. 次にやること
3. 最後にやること

> 出題時には自動でシャッフルされます。

## 正誤問題は「答え:」の行を書く(この文が正しいか問う)

答え: ○

> ○/×のほか o, x, 正, 誤, true, false でもOK。

## 問題文の補足・コード・画像も書ける

見出しの下に書いた段落は問題文の続きとして表示されます。

```javascript
console.log("コードブロックもOK");
```

- [x] 選択肢には **太字** や `コード` も使える
- [ ] 画像は `![alt](url)` で挿入できる

> 解説も複数行・Markdown記法OKです。
````

### 判定ルール早見表

| 書き方 | 形式 |
| --- | --- |
| `- [x]` / `- [ ]` のチェックリスト | 選択問題(`[x]`が1つ=択一、2つ以上=複数選択) |
| `1.` `2.` … の番号付きリスト | 並び替え(書いた順=正解) |
| `答え: ○` / `答え: ×` の行 | 正誤(○×)問題 |
| `>` の引用 | 解説(回答後に表示) |

形式を判別できない問題はスキップされ、ブラウザの開発者コンソールに警告が出ます。

## ファイル構成

```
index.html              アプリ本体(SPA)
css/style.css           スタイル
js/parser.js            Markdown → 問題データ変換
js/storage.js           成績・間違い履歴の保存(localStorage)
js/app.js               出題・判定・画面遷移
data/index.json         デッキ一覧(ここに md を登録)
data/*.md               問題データ
.github/workflows/      GitHub Pages 自動デプロイ
```
